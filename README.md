# :car: Vehicle Sales

## :octocat: the Clone project
```shell
git@github.com:LuizaAlanis/vehicle-sales.git
```

## :art: Run front-end

On Client folder...

```shell
cd client
```

Run the command:
```shell
npm start
```

Then open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## :game_die: Run back-end

On Server folder...

```shell
cd server
```

Run the command:
```shell
npm run dev start
```
